## 🕹️ Lobby & Multiplayer

<Steps>
  <Step title="Server erstellen">
    Hauptmenü → **New Store** → Name \+ Sichtbarkeit (Private / Only Friends / Public) \+ Schwierigkeit → **Create**.
  </Step>
  <Step title="Freunde einladen">
    Im Spiel `Esc` → **Copy Join Code to Clipboard** → Code an Freunde schicken.
  </Step>
  <Step title="Beitreten">
    Hauptmenü → **Join Store** → Code einfügen oder Public-Server suchen.
  </Step>
</Steps>

<Warning>
  Gebt **Manager-Rechte** nur an vertraute Spieler – Trolle können sonst euer Geld komplett verheizen.
</Warning>

## 📋 Tagesablauf

<Steps>
  <Step title="Tag öffnen">
    Stelle Kasse besetzen, Türen öffnen lassen.
  </Step>
  <Step title="Kunden bedienen">
    Produkte scannen, Preis kassieren, Wechselgeld zurückgeben (aufgerundete Preise helfen!).
  </Step>
  <Step title="Auffüllen, wenn ruhig">
    Erst leere Regale füllen, dann neue Ware bestellen. **Nie ein leeres Regal stehen lassen.**
  </Step>
  <Step title="Diebe abwehren (ab Tag ~8)">
    Diebe haben **\$-Icon** über dem Kopf → mit dem **Besen** schlagen, bis Symbol weg ist.
  </Step>
  <Step title="Tag beenden">
    Boxen recyceln (über die Straße, \+1,50 \$/Box), Bilanz checken.
  </Step>
  <Step title="Donnerstag = Inflation">
    Preise aller betroffenen Kategorien anpassen, sonst Verlust.
  </Step>
</Steps>

## 👥 Rollenverteilung (Co-op)

| Rolle | Aufgabe |
| --- | --- |
| Kassierer:in | Kasse, Wechselgeld, Diebe-Spotter |
| Restocker:in | Regale auffüllen, Lager organisieren |
| Manager:in | Bestellen, Preise anpassen, Franchise Points |
| Security | Diebe verprügeln, Türen sichern |

<Tip>
  Schließe **niemals die einzige Kasse**, während der Laden offen ist – Kunden gehen dann mit Ware raus, ohne zu zahlen.
</Tip>
