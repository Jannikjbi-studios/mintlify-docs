## 🎮 Steuerung (typisch)

| Aktion | Taste |
| --- | --- |
| Schub vor / zurück | `W` / `S` |
| Ruder links / rechts | `A` / `D` |
| Bugstrahlruder (Bow Thruster) | `Q` / `E` |
| Anker | `B` |
| Funk / Karte | `R` / `M` |
| Kamera wechseln | `C` |
| Wasserkanone / Crane | Maus \+ `E` |
| Aus-/Einsteigen (zu Fuß auf Deck) | `F` |

## ⚓ Ablegen & Anlegen

<Steps>
  <Step title="Vor Abfahrt">
    Motor starten, Festmacher / Leinen lösen, Schubrichtung prüfen.
  </Step>
  <Step title="Vom Steg lösen">
    Erst mit **Bugstrahlruder** parallel wegschieben, dann sanft Hauptmaschine.
  </Step>
  <Step title="Im Hafen">
    Max. langsame Fahrt (oft ≤ 5 kn) – sonst Strafe / Schaden.
  </Step>
  <Step title="Anlegen">
    Flach im Winkel anfahren, kurz vorm Kontakt parallel ausrichten, mit Bugstrahlruder andocken.
  </Step>
  <Step title="Festmachen">
    Crew bzw. Interaktion an Pollern, Mission abschließen.
  </Step>
</Steps>

| Fraktion | Typische Aufgaben |
| --- | --- |
| Cargo / Logistik | Container, Bulk-Cargo, LNG-Transport |
| Tide Guard (Coast Guard) | Patrouillen, Search & Rescue, Schlepper-Einsätze |
| Firefighting | Brände auf See mit Wasserkanonen löschen |
| Salvage | Verlorene Fracht bergen |

## 🌊 Wetter & Wellen

- Unreal Engine 5 \+ NVIDIA WaveWorks 2.0 → echte Wellenphysik.
- Bei Sturm Geschwindigkeit reduzieren, Crew-Anweisungen befolgen.
- Nebel: Radar & Funk aktiv halten.

<Warning>
  Niemals **Vollschub rückwärts im Hafen** – Schadensrisiko und Mission ggf. fehlgeschlagen.
</Warning>
