## 🎮 Steuerung Pistenraupe (Pistenbully / Leitwolf)

| Aktion | Taste |
| --- | --- |
| Gas / Rückwärts | `W` / `S` |
| Lenken | `A` / `D` |
| Frontschild heben / senken | `R` / `F` |
| Frontschild neigen | Pfeiltasten |
| Fräse hoch / runter | `T` / `G` |
| Seilwinde aus- / einrollen | `Y` / `H` |
| Differenzialsperre | `X` |
| Lichter | `L` |
| Aussteigen | `F` (am Fahrzeug) |

## 🚠 Seilbahn in Betrieb nehmen

<Steps>
  <Step title="Talstation aufschließen">
    Hauptschalter / Sicherungsraum aktivieren.
  </Step>
  <Step title="Connect-Steuerung starten">
    Authentisches **Doppelmayr Connect**-Panel: Selbsttest abwarten.
  </Step>
  <Step title="Antrieb hochfahren">
    Geschwindigkeit langsam steigern, Sensoren prüfen.
  </Step>
  <Step title="Sessel / Gondeln einsteigen lassen">
    Erst freigeben, wenn alle Anzeigen grün sind.
  </Step>
  <Step title="Bei Stop">
    Notstopp drücken, Ursache prüfen (z. B. gestürzter Gast), dann manuell weiterfahren.
  </Step>
</Steps>

## 🎿 Tagesablauf

<Steps>
  <Step title="Morgens präparieren">
    Pisten vor Saisonstart fräsen und glätten. **Steile Pisten = Seilwinde verwenden**.
  </Step>
  <Step title="Beschneien (bei Bedarf)">
    TR-10 Schneekanone auf passenden Pisten platzieren, Wasserleitung anschließen.
  </Step>
  <Step title="Lifte starten">
    Sicherheitscheck → Connect-Panel → Lift in Betrieb.
  </Step>
  <Step title="Missionen erledigen">
    Lieferaufträge, Personentransport, Aushilfen im Resort.
  </Step>
  <Step title="Feierabend">
    Lifte herunterfahren, Pistenraupen einsetzen für nächsten Tag.
  </Step>
</Steps>

<Warning>
  **Seilwinde nicht senkrecht oder über Felsen führen** – sonst reißt das Seil. Immer flach am Pisten-Anker einhängen.
</Warning>
