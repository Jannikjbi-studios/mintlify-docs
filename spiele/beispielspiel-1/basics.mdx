## 🎮 Steuerung

| Aktion | Tastatur | Controller |
| --- | --- | --- |
| Bewegen | `W A S D` | Linker Stick |
| Sprinten | `Shift` | L3 |
| Springen | `Leertaste` | A / X |
| Angriff | Linksklick | RB / R1 |
| Block | Rechtsklick | LB / L1 |

## 🕹️ Erste Schritte im Spiel

<Steps>
  <Step title="Charakter erstellen">
    Wähle eine Klasse, die zu deinem Spielstil passt.
  </Step>
  <Step title="Tutorial abschließen">
    Das Tutorial schaltet wichtige Fähigkeiten frei – nicht überspringen!
  </Step>
  <Step title="Erste Region erkunden">
    Sammle Ressourcen und schalte Schnellreise-Punkte frei.
  </Step>
