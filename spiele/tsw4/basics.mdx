## 🎮 Steuerung (Standard)

| Aktion | Taste |
| --- | --- |
| Beschleunigen | `A` (Fahrhebel \+) |
| Bremsen | `D` (Fahrhebel –) |
| Türen öffnen/schließen | `Ü` / `+` |
| Pfeife / Horn | `Leertaste` |
| Lichter | `L` |
| Führerstand-Kamera | `1` |
| Außen-Kamera | `2` |
| PZB / SIFA quittieren | `Bild ab` / `End` |

## 🚂 Erste Fahrt

<Steps>
  <Step title="Lok einrichten">
    Stromabnehmer hoch, Hauptschalter ein, Richtungswender vor.
  </Step>
  <Step title="Bremse lösen">
    Führerbremsventil lösen, Federspeicher lösen.
  </Step>
  <Step title="Signal beachten">
    Erst losfahren, wenn das Signal Fahrt zeigt!
  </Step>
  <Step title="PZB beachten">
    Beim Vorsignal `1000 Hz` mit `Bild ab` quittieren.
  </Step>
</Steps>

<Warning>
  Verpasste PZB-Quittierungen führen zur Zwangsbremsung – immer aufmerksam bleiben!
</Warning>
